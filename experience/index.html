My Cloud Experience

Welcome to my cloud journey! Below is an overview of my experience, focusing primarily on AWS and cloud technologies. As I build my career in cloud computing, I continue to grow my skills and work on exciting projects. Here's a breakdown of my experience:

Cloud Support Engineer Apprenticeship

In my apprenticeship, I gained hands-on experience with AWS cloud support. I learned troubleshooting techniques for cloud-related issues, helped customers with configuring and managing AWS services, and provided guidance on best practices for AWS security and performance optimization.

Key Cloud Technologies Used

  • AWS S3 – Used for static website hosting and file storage.
  • AWS Route 53 – Managed DNS configurations for custom domains.
  • AWS CloudFront – Used to distribute content globally via a Content Delivery Network (CDN).
  • AWS ACM – Managed SSL/TLS certificates to secure websites.
  • AWS Lambda – Explored serverless computing for efficient and scalable applications.

Current Projects

Currently, I'm working on integrating an appointment booking system with Google Calendar, built using serverless infrastructure. I'm using AWS Lambda for back-end logic and API Gateway to create a smooth, event-driven application. Additionally, I plan to integrate more advanced cloud solutions such as containerization and automation pipelines to further enhance my skills in cloud architecture.

Technical Skills

  • Cloud Infrastructure – Proficient in deploying and managing AWS infrastructure using services such as EC2, S3, and Route 53.
  • Security – Focused on cloud security best practices, including IAM roles, security groups, and encryption in transit and at rest.
  • Automation – Experience with AWS CloudFormation and other automation tools to streamline deployments and management.
  • Development – Building full-stack applications with front-end frameworks and AWS back-end services (e.g., Lambda, DynamoDB, API Gateway).

Why Cloud?

I’m passionate about cloud computing because it offers unlimited scalability, security, and the ability to solve complex real-world problems. The flexibility of cloud-native technologies allows me to create innovative solutions that help businesses scale and innovate quickly. I'm excited to continue my journey in cloud computing and explore more advanced cloud architecture techniques.
