About Me

Hey! I’m Nick. Thanks for visiting my first major project! Before you stands a determined learner with a sharp focus on Cloud technologies and a deep appreciation for solving real-world problems. I’m transitioning into cloud architecture and security, and have built this portfolio entirely on AWS, using many services like S3, Route 53, CloudFront, ACM, and Lambda.

My background includes a mix of IT support, pre-sales experience, and a recent AWS Cloud Support apprenticeship. I enjoy blending technical implementation with user-focused design — hence this growing portfolio!

Right now, I’m working on integrating an appointment booking system with Google Calendar, all built using serverless infrastructure. I love learning by building, and my curiosity drives everything — especially when it comes to cloud-native projects.
